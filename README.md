# Cryptographic Hardware Accelerators for RISC-V

This project focuses on the design and implementation of cryptographic hardware accelerators developed in **Chisel** and integrated into the **Rocket Chip RISC-V** processor pipeline. 

### Key Objectives:
* **Standard Alignment:** Implementation of instruction set extensions compliant with the **RISC-V Scalar Cryptography (Zk)** standard.
* **Optimization:** Enhancing hardware for maximum throughput and area efficiency on FPGA targets.
* **Security Hardening:** Implementing side-channel countermeasures, including **LFSR-based power blurring**, to mitigate data-dependent power leakage.
* **Evaluation:** Benchmarking hardware speedups over software implementations and conducting comprehensive resource utilization and security analysis.
